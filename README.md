## ⚡ Run Locally (3 steps)

```bash
pip install -r requirements.txt
cp .env.example .env        # Add your API key inside .env
python app.py               # Open http://localhost:5000
```

## 🔑 Get API Key (Free)
→ console.anthropic.com → Sign up → API Keys → Create Key

## ☁️ Deploy FREE on Render.com
1. Push to GitHub
2. render.com → New Web Service → connect repo
3. Add env vars: ANTHROPIC_API_KEY, SECRET_KEY, SITE_URL
4. Deploy → Live at sunai.onrender.com ✅
